# python-data-app-template
包含 Python、Pandas、Plotly、Streamlit 與 AI 輔助的開發環境範本

## 就業金卡報表應用

專案目前提供兩種前端：

- `就業金卡/app.py`：原本的 Streamlit 版本
- `就業金卡_dash/app.py`：新的 Plotly Dash 版本（含 21 份報表）

## 快速啟動

```bash
pip install -r requirements.txt
```

Streamlit:

```bash
streamlit run 就業金卡/app.py
```

Plotly Dash:

```bash
python 就業金卡_dash/app.py
```
